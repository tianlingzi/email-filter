# EmailFilter 插件

## 插件介绍

EmailFilter 是一款专为 Typecho 1.2.1 设计的邮箱过滤插件，用于在用户注册过程中过滤特定的邮箱地址。该插件可以有效阻止来自不受信任域名或特定邮箱地址的用户注册，提高网站用户质量和安全性。

## 功能特点

- **灵活的邮箱过滤规则**：支持精确匹配和通配符匹配两种模式
- **自定义错误提示**：可自定义用户使用被过滤邮箱时的提示信息
- **用户友好的配置界面**：直观的设置页面，易于配置和管理
- **版本兼容性检查**：确保插件在兼容的 Typecho 版本上运行
- **完善的错误处理**：包含异常捕获和日志记录功能

## 安装说明

1. **下载插件**：将插件文件下载或克隆到本地
2. **上传插件**：将 `Plugin.php` 文件上传到 Typecho 网站的 `usr/plugins/EmailFilter` 目录下
3. **启用插件**：登录 Typecho 后台，进入「控制台」→「插件」，找到「EmailFilter」插件并点击「启用」
4. **配置插件**：启用后点击「设置」按钮，配置需要过滤的邮箱地址

## 使用方法

### 配置过滤规则

1. 在插件设置页面中，在「过滤邮箱列表」文本框中输入需要过滤的邮箱地址，每行一个
2. 支持两种匹配模式：
   - **精确匹配**：例如 `1245@test.com`，只会过滤这个特定的邮箱地址
   - **通配符匹配**：例如 `*@test.com`，会过滤 test.com 域名下的所有邮箱地址

### 自定义错误消息

在「错误提示信息」文本框中，可以自定义当用户使用被过滤的邮箱注册时显示的错误消息。默认为「该邮箱未在信任范围内，请更换其他邮箱」。

## 配置示例

以下是一些配置示例：

```
# 精确匹配特定邮箱
1245@test.com
spam@example.org
admin@example.com

# 通配符匹配整个域名
*@test.com
*@spamdomain.org
*@temp-mail.org
```
